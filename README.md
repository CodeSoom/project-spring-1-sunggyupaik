## Bookclub

IT 서적 스터디와 공유를 위한 플랫폼입니다.  
자신이 원하는 책을 주제로 팀원들과 스터디 할 수 있습니다.

## 기술 스택

- Java 11
- Spring Boot 2.5.x
- Gradle
- Spring Data Jpa
- QueryDSL
- Spring Security
- Rabbitmq
- Elastic Search
- Redis
